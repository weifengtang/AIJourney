## 一、核心数据结构

### 1.1 Message（消息）

```python
@dataclass
class Message:
    role: str                    # user / assistant
    content: str                 # 消息内容
    timestamp: datetime          # 时间戳
    tool_calls: List[Dict[str, Any]] = []  # 工具调用（可选）
```

**字段说明**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| role | str | ✅ | 消息角色：`user`（用户）或 `assistant`（AI） |
| content | str | ✅ | 消息文本内容 |
| timestamp | datetime | ✅ | 消息时间戳（ISO 8601） |
| tool_calls | List[Dict] | ❌ | 工具调用记录（Claude Code 等场景） |

### 1.2 SessionData（会话数据）

```python
@dataclass
class SessionData:
    session_id: str              # 会话唯一标识
    source: str                  # 来源
    project_path: Optional[str]  # 项目路径
    start_time: datetime         # 开始时间
    end_time: Optional[datetime] # 结束时间
    title: Optional[str]         # 会话标题
    summary: Optional[str]       # 摘要
    messages: List[Message]      # 消息列表
    files_modified: List[str]    # 修改的文件列表
    tokens_input: int            # 输入 token 数
    tokens_output: int           # 输出 token 数
```

**字段说明**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| session_id | str | ✅ | 会话唯一标识（UUID 或平台 ID） |
| source | str | ✅ | 数据来源：`claude_code` / `codebuddy` / `vscode` / `idea` / `doubao_selenium` |
| project_path | str | ❌ | 关联的项目绝对路径 |
| start_time | datetime | ✅ | 会话开始时间 |
| end_time | datetime | ❌ | 会话结束时间 |
| title | str | ❌ | 会话标题（从内容或平台提取） |
| summary | str | ❌ | 会话摘要文本 |
| messages | List[Message] | ✅ | 消息列表 |
| files_modified | List[str] | ❌ | 修改的文件路径列表 |
| tokens_input | int | ✅ | 输入 Token 数量 |
| tokens_output | int | ✅ | 输出 Token 数量 |

### 2.3 报告周期枚举

```python
class ReportPeriod(Enum):
    DAILY = "daily"      # 日报
    WEEKLY = "weekly"    # 周报
    MONTHLY = "monthly"  # 月报
    YEARLY = "yearly"    # 年报
```

**日期范围规则**：

| 周期 | 起始日期 | 结束日期 | 文件名格式 |
|------|----------|----------|------------|
| daily | target_date | target_date | `YYYYMMDD` |
| weekly | 本周一 | 本周日 | `YYYYMMDD-MMDD` |
| monthly | 本月1日 | 本月最后一天 | `YYYYMM` |
| yearly | 本年1月1日 | 本年12月31日 | `YYYY` |
