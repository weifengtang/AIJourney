# AI Journey - OpenClaw Skill

AI 对话历史记录与分析工具，支持 Claude Code、CodeBuddy、Git 数据源。

## 安装

```bash
skillhub install ai_journey
```

## 使用

```bash
# 日报
python3 scripts/daily-report.py

# AI 增强日报
python3 scripts/daily-report-ai.py

# 周报
python3 scripts/weekly-report.py

# 区间报告
python3 scripts/range-report.py --week
```
